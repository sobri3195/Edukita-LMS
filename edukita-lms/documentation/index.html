EduKita-LMS

HTML5 LMS Template — ThemeForest Ready

1. Panduan Instalasi

  1. Ekstrak paket ZIP yang Anda unduh.
  2. Folder utama template: /edukita-lms/.
  3. Buka file index.html di peramban untuk melihat demo.
  4. Unggah seluruh folder edukita-lms ke server/hosting Anda.

3. Petunjuk Kustomisasi

  • Warna dan tipografi diatur pada css/style.css (variabel CSS --ek-*).
  • SCSS sumber tersedia di scss/style.scss untuk workflow build Anda.
  • Komponen berbasis Bootstrap 5 — rujuk dokumentasi Bootstrap untuk utilitas tambahan.
  • Gambar hero dan kursus dapat diganti di folder images/.
  • Terjemahan dasar ID/EN menggunakan atribut data-i18n dan skrip di js/main.js.
  • Formulir menggunakan kelas ajax-ready untuk submit AJAX (mock/demo).

Contoh mengubah warna utama:

:root{
  --ek-primary:#2B7A78; /* ganti sesuai brand Anda */
}

4. Optimasi & SEO

  • Setiap halaman memiliki meta description, keywords, dan og:image.
  • Gunakan format gambar terkompresi (JPG/PNG). File preview telah dikompresi.
  • Pastikan Anda melakukan minify pada CSS/JS sebelum produksi jika perlu.

5. Lisensi & Kredit Aset

  • Font: Google Fonts — Poppins, Open Sans (free for commercial use).
  • Ikon: Font Awesome (free) — pastikan lisensi sesuai kebutuhan Anda.
  • Gambar: Placeholder dibuat secara programatik untuk keperluan demo (bebas digunakan).
  • Library: Bootstrap 5 (MIT), AOS (MIT).

Mohon cek ulang lisensi pihak ketiga sebelum produksi.

6. Dukungan

Jika Anda membeli template ini melalui ThemeForest, gunakan saluran dukungan resmi untuk bantuan teknis.
