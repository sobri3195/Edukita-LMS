EduKita-LMS

HTML5 Responsive LMS Template

1. Panduan Instalasi

  1. Ekstrak paket edukita-lms ke folder server/hosting Anda.
  2. Pastikan koneksi internet tersedia untuk CDN (Bootstrap, Font Awesome, AOS). Jika ingin offline, unduh aset lokal dan perbarui path.
  3. Buka index.html di browser untuk melihat halaman utama.

3. Petunjuk Kustomisasi

  • Warna & gaya: Ubah variabel CSS di css/style.css atau edit sumber SCSS di scss/style.scss.
  • Font: Template menggunakan Google Fonts (Poppins & Open Sans) di css/style.css.
  • Komponen: Sesuaikan section pada file HTML terkait (landing, course, blog, dashboard).
  • Animasi: AOS dapat diaktifkan/diatur di index.html (lihat AOS.init).
  • Form: Form sudah AJAX-ready untuk demo. Ganti logika di js/main.js dengan endpoint backend Anda.
  • Multi-bahasa: Contoh switcher ada di navbar. Tambahkan entri translations di js/main.js sesuai kebutuhan.
  • SEO: Perbarui meta title, description, keywords, dan og:image setiap halaman.

4. Lisensi & Kredit Aset

  • Framework UI: Bootstrap 5 (MIT)
  • Ikon: Font Awesome (Free)
  • Font: Google Fonts (Poppins, Open Sans)
  • Gambar contoh/preview: Silakan ganti dengan gambar dari Unsplash/Pexels (bebas pakai komersial). Saat ini menggunakan placeholder.

Catatan: Pastikan mengganti gambar placeholder sebelum rilis ke marketplace.
